# KIOS GAME RENTAL PLAYSTATION

Website ini dibuat sebagai bagian dari Exercise 1 Pengembangan Desain Web.  
Tema yang dipilih adalah **Rental PlayStation Kios Game** yang berlokasi di Purbalingga, Jawa Tengah.

## 🎮 Deskripsi
Kios Game menghadirkan pengalaman bermain konsol PS5 terbaru dengan fasilitas nyaman, ruangan ber-AC, kursi gaming ergonomis, dan koleksi game AAA terupdate.  
Website ini menampilkan informasi tentang:
- Home (promosi utama)
- Tentang Kami (profil dan misi Kios Game)
- Fasilitas (peralatan dan kenyamanan yang tersedia)
- Lokasi (alamat lengkap dengan Google Maps)

## ✨ Fitur
- Navigasi antar section dengan scroll halus
- Hero section dengan tombol **Daftar Member**
- Alert promo ketika tombol ditekan
- Galeri fasilitas (PS5, controller)
- Embed Google Maps untuk lokasi
- Footer dengan logo Instagram
